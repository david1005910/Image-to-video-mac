🎬 Midjourney Video Creator 사용법 & 테스트

⚡ 빠른 시작

1️⃣ Extension 설치 완료 확인 (브라우저 상단 아이콘)

2️⃣ 이 페이지에서 테스트 이미지로 연습

3️⃣ Discord/Midjourney에서 실제 사용

📖 사용법 가이드

Step 1: Extension 활성화 확인

✅ 브라우저 우측 상단에 Extension 아이콘이 보이나요?

✅ 이 페이지 우측 하단에 🎬 플로팅 버튼이 보이나요?

Step 2: 플로팅 버튼 사용법

페이지 우측 하단의 🎬 버튼을 클릭하면 컨트롤 패널이 열립니다.

패널 기능:

  • 📸 이미지 수집 시작/중지
  • 🎬 동영상 생성
  • 🗑️ 수집 목록 초기화
  • ⚙️ 설정 변경

Step 3: 이미지 수집

자동 수집: "자동 수집" 체크 시 페이지의 모든 이미지 자동 감지

수동 수집: "이미지 수집 시작" 클릭 후 원하는 이미지가 있는 곳 스크롤

Upscale 필터: 고품질 이미지만 수집하려면 "Upscale만 수집" 체크

Step 4: 동영상 생성

1. 충분한 이미지 수집 (최소 3-5개 권장)

2. "🎬 동영상 생성" 버튼 클릭

3. 로컬 서버에서 처리 (약 1-3분)

4. 완료 시 다운로드 링크 표시

🎯 실제 사용 시나리오

Discord에서 사용하기

1. https://discord.com 접속 2. Midjourney 봇이 있는 서버/채널 이동 3. /imagine 명령어로 이미지 생성 4. Extension으로 이미지 수집 5. 동영상 생성

Midjourney.com에서 사용하기

1. https://www.midjourney.com 접속 2. 로그인 후 Gallery 이동 3. Extension으로 이미지 수집 4. 동영상 생성

💡 사용 팁

  • 최적 이미지 수: 5-10개가 적당 (너무 많으면 처리 시간 증가)
  • 이미지 품질: Upscale된 이미지 사용 권장
  • 테마 일관성: 비슷한 스타일의 이미지로 통일감 있는 영상 제작
  • 자동 스크롤: 많은 이미지 수집 시 자동 스크롤 기능 활용

⚠️ 주의사항

  • 로컬 서버(localhost:3000)가 실행 중이어야 합니다
  • 이미지 저작권을 확인하세요
  • 브라우저 메모리 관리를 위해 주기적으로 수집 목록을 초기화하세요

🔧 문제 해결

플로팅 버튼이 안 보일 때

1. 페이지 새로고침 (F5 또는 Cmd+R)

2. Extension 아이콘 클릭 → 팝업에서 상태 확인

3. chrome://extensions/ 에서 Extension 새로고침

이미지 수집이 안 될 때

1. "수집 시작" 버튼 다시 클릭

2. 이미지가 완전히 로드될 때까지 대기

3. 자동 수집 옵션 체크

동영상 생성 실패 시

1. 로컬 서버 실행 확인 (http://localhost:3000)

2. 수집된 이미지가 1개 이상인지 확인

3. 네트워크 연결 상태 확인
